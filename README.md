국방부 퀘스트를 마치고 학교로 돌아왔더니 모범적인 학술 동아리가 되어있었다?



동아리의 변화를 이끌어낸 선배

새로워진 동아리에 합류하는 신입생

나도 모르는 사이 내 일상을 파고드는 두 사람!
